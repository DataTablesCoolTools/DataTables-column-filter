DataTables Column Filter Filters in header.

Column filters can be placed in the header instead of the footer. You ned to add sPlaceHolder property with value head:after, or head:before. Note that you need to manually put two table rows in the THEAD section of the table. One table heading will be used by data tables for sorting, and the other will be used for filtering. In this example is shown filter header row placed after datatablses header. Alternative postion is shown in Date/number ranges page.
